CMake Integration

Simple yet powerful CMake integration extension for Visual Studio Code.

Install Extension

Simple

Just install the extension, open as many workspace folders as you like and start building your projects with the predefinied configurations.

Powerful

Create configurations to fully customize your build process including build directory, toolchain file, cache entries and more. Specify dependencies between workspace folders to create a advanced build flow.

Integrated

Control and visualize every aspect of your build within Visual Studio Code like the target selection, compiler diagnostics and more.
